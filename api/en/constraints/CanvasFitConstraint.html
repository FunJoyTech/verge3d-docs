[name]

A constraint allows to attach panels or texts where you want whem to be, regardless of screen resolution or device. Have to parent the object to the camera.

Constraints are used to set up various relations between objects by affecting their position, rotation and scale. Can be used with the [page:Object3D.constraints Object3D.constraints] property.

[name]([page:Object3D target])

Properties

See the base [page:TargetConstraint] class for common properties.

[property:Number edgeH]

The horizontal canvas edge to fit to. Default is *0*.
-1 — left edge.
1 — right edge.
0 — none egde.

[property:Number edgeV]

The horizontal canvas edge to fit to. Default is *0*.
-1 — bottom edge.
1 — top edge.
0 — none egde.

[property:Float offset]

Canvas fir offset, must be >= 0. Default is *0.0*.

[property:Boolean isCanvasFitConstraint]

Used to check whether this or derived classes are canvas fit constraints. Default is *true*.

[property:String type]

The constraint's type. Equals to "CanvasFitConstraint".

[property:Boolean useBoundingBox]

Canvas fit shape. Default is *true*.
false — sphere.
true — box.
