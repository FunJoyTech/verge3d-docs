[name]

A class containing useful rendering methods.

Methods

[method:WebGLCubeRenderTarget renderSceneToCubemap]([param:Scene scene], [param:WebGLRenderer renderer], [param:WebGLCubeRenderTarget cubeRenderTarget], [param:Object cameraOptions])

Render the given scene into a cubemap. The parameters are:
  • scene - [page:Scene] to render into a cubemap
  • renderer - [page:WebGLRenderer]
  • cubeRenderTarget - [page:WebGLCubeRenderTarget] render target containing the cubemap to render into
  • cameraOptions - an object containing options for the cube camera used to render the scene into the cubemap; the options and their default values are: { camPos: new Vector3(), clipStart: 0.1, clipEnd: 100, } camPos - cube camera position
    clipStart - near plane value
    clipEnd - far plane value

[method:WebGLCubeRenderTarget renderWorldNodeMatToCubemap]([param:WebGLRenderer renderer], [param:MeshNodeMaterial material], [param:Integer cubeRes], [param:Object options])

Render a world node material to a cubemap. The parameters are:
  • renderer - [page:WebGLRenderer]
  • material - world node material, should be of type [page:MeshNodeMaterial]
  • cubeRes - resolution of the rendered cubemap
  • options - (optional) object that holds texture parameters for an auto-generated cube render target; has the same format as the *options* parameter for the [page:WebGLCubeRenderTarget] constructor
The resulting [page:WebGLCubeRenderTarget render target] with a cube texture in it can be used as a scene background.
