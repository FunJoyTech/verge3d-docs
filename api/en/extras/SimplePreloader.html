[name]

Implements simple rotating preloader. Check out the Verge3D Player application to see it in action.

Constructor

[name]([page:Object parameters])

[page:Object parameters] - object with properties defining the preloader's behavior. All parameters except container are considered optional. The following are valid parameters:
[page:DOMElement container] - A container element or container element ID used by the created preloader.
[page:Number imageRotationSpeed] - image rotation speed (degrees per second).

Methods

Simple preloader overrides [page:Preloader Preloader] methods implementing image animation and cleanup logic.
