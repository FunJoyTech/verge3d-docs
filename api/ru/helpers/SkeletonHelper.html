[page:Object3D] → [page:Line] → [page:LineSegments] →

[name]

Вспомогательный объект, помогающий визуализировать [page:Skeleton Skeleton]. Помощник визуализируется с помощью [page:LineBasicMaterial LineBasicMaterial].

Пример Кода

const helper = new v3d.SkeletonHelper(skinnedMesh); scene.add(helper);

Примеры

[example:webgl_animation_skinning_blending WebGL / animation / skinning / blending]
[example:webgl_animation_skinning_morph WebGL / animation / skinning / morph]
[example:webgl_loader_bvh WebGL / loader / bvh]

Constructor

[name]([param:Object3D object])

object -- Обычно это экземпляр [page:SkinnedMesh]. Однако можно использовать любой экземпляр [page:Object3D], если он представляет собой иерархию [page:Bone Bone]ов (через [page:Object3D.children]).

Свойства

[property:Array bones]

Список костей, которые помощник отображает как [page:Line Lines].

[property:Object3D root]

Объект, переданный в конструкторе.
