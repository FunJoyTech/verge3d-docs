[page:Object3D] →

[name]

Создает визуального помощника, состоящего из сферического [page:Mesh] для [page:HemisphereLight HemisphereLight].

Пример Кода

const light = new v3d.HemisphereLight(0xffffbb, 0x080820, 1); const helper = new v3d.HemisphereLightHelper(light, 5); scene.add(helper);

Конструктор

[name]([param:HemisphereLight light], [param:Number sphereSize], [param:Hex color])

[page:HemisphereLight light] -- Визуализируемый свет.

[page:Number size] -- Размер меша, используемой для визуализации света.

[page:Hex color] -- (опционально) если этот параметр не установлен, помощник примет цвет освещения.

Свойства

Общие свойства см. в базовом классе [page:Object3D].

[property:HemisphereLight light]

Референс на визуализируемый HemisphereLight.

[property:Object matrix]

Референс на hemisphereLight's [page:Object3D.matrixWorld matrixWorld].

[property:Object matrixAutoUpdate]

См. [page:Object3D.matrixAutoUpdate]. Здесь установлено значение *false*, так как помощник использует hemisphereLight's [page:Object3D.matrixWorld matrixWorld].

[property:hex color]

Параметр цвета, передаваемый в конструкторе. По умолчанию *undefined*. Если этот параметр изменить, цвет помощника будет обновлен при следующем вызове [page:.update update].

Методы

Общие методы см. в базовом классе [page:Object3D].

[method:null dispose]()

Утилизирует hemisphereLightHelper.

[method:null update]()

Обновляет помощника в соответствии с положением и направлением [page:.light].
