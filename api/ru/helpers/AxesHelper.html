[name]

Объект оси для визуализации 3 осей простым способом.
Ось X имеет красный цвет. Ось Y - зеленый. Ось Z - синий.

Пример Кода

const axesHelper = new v3d.AxesHelper(5); scene.add(axesHelper);

Примеры

[example:webgl_buffergeometry_compression WebGL / buffergeometry / compression]
[example:webgl_geometry_convex WebGL / geometry / convex]
[example:webgl_loader_nrrd WebGL / loader / nrrd]

Конструктор

[name]([param:Number size])

[page:Number size] -- (опционально) размер линий, представляющих оси. По умолчанию *1*.
