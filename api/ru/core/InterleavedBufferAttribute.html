[name]

Конструктор

[name]([param:InterleavedBuffer interleavedBuffer], [param:Integer itemSize], [param:Integer offset], [param:Boolean normalized])

Свойства

[property:InterleavedBuffer data]

Экземпляр [page:InterleavedBuffer InterleavedBuffer], переданный в конструкторе.

[property:TypedArray array]

Значение [page:InterleavedBufferAttribute.data data].array.

[property:Integer count]

Значение параметра [page:InterleavedBufferAttribute.data data].count. Если в буфере хранится трехкомпонентный элемент (например, позиция, нормаль или цвет), то подсчитывается количество таких элементов.

[property:Integer itemSize]

Сколько значений составляет каждый элемент.

[property:String name]

Необязательное имя для данного экземпляра атрибута. По умолчанию это пустая строка.

[property:Boolean needsUpdate]

По умолчанию *false*. Если установить значение *true*, то весь чередующийся буфер (а не только конкретные данные атрибутов) будет снова отправлен на GPU.

[property:Boolean normalized]

По умолчанию *false*.

[property:Integer offset]

Смещение в буфере базового массива, с которого начинается элемент.
