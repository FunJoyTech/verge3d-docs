[name]

"Interleaved" означает, что несколько атрибутов, возможно, разных типов (например, положение, нормаль, uv, цвет), упакованы в один буфер массива.

Введение в чередующиеся массивы можно найти здесь: [link:https://blog.tojicode.com/2011/05/interleaved-array-basics.html Interleaved array basics]

Примеры

[example:webgl_buffergeometry_points_interleaved webgl / buffergeometry / points / interleaved]

Конструктор

[name]([param:TypedArray array], [param:Integer stride])

[page:TypedArray array] -- Типизированный массив с общим буфером. Хранит геометрические данные.
[page:Integer stride] -- Количество элементов типизированного массива на вершину.

Свойства

[property:Array array]

Типизированный массив с общим буфером. Хранит геометрические данные.

[property:Integer stride]

Количество элементов типизированного массива на вершину.

[property:Integer count]

Дает общее количество элементов в массиве.

[property:Object updateRange]

Объект, содержащий смещение и количество.

[property:Number updateRange.offset]

По умолчанию *0*.

[property:Number updateRange.count]

По умолчанию *-1*.

[property:String uuid]

[link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID] данного экземпляра. Он присваивается автоматически, поэтому его не следует редактировать.

[property:Integer version]

Номер версии, увеличивающийся каждый раз, когда свойство needsUpdate установлено в true.

[property:Boolean needsUpdate]

По умолчанию *false*. Установка этого значения в true увеличивает [page:InterleavedBuffer.version version].

[property:Usage usage]

Определяет предполагаемую схему использования хранилища данных в целях оптимизации. Соответствует параметру *usage* в [link:https://developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/bufferData WebGLRenderingContext.bufferData]().

Методы

[method:InterleavedBuffer copy]([param:InterleavedBuffer source])

Копирует другой [name] в этот [name].

[method:InterleavedBuffer copyAt]([param:Integer index1], [param:InterleavedBuffer attribute], [param:Integer index2])

Копирует данные из атрибута[index2] в [page:InterleavedBuffer.array array][index1].

[method:InterleavedBuffer set]([param:TypedArray value], [param:Integer offset])

value - Исходный (типизированный) массив.
offset - Смещение в целевом массиве, с которого следует начать запись значений из исходного массива. По умолчанию *0*.

Сохраняет несколько значений в буфере, считывая входные значения из указанного массива.

[method:InterleavedBuffer clone]([param:Object data])

data - Этот объект содержит буферы общего массива, необходимые для правильного клонирования геометрии с чередующимися атрибутами.

Создает клон данного [name].

[method:InterleavedBuffer setUsage] ([param:Usage value])

Установите [page:InterleavedBuffer.usage usage] в значение.

[method:InterleavedBuffer toJSON]([param:Object data])

data - Этот объект содержит буферы общего массива, необходимые для правильной сериализации геометрии с чередующимися атрибутами.

Сериализует этот [name].
