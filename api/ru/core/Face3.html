[name]

Треугольная грань, используемая в [page:Geometry]. Они создаются автоматически для всех стандартных типов геометрии, однако если вы создаете пользовательскую геометрию, вам придется создавать их вручную.

Пример Кода

const material = new v3d.MeshStandardMaterial({ color : 0x00cc00 }); //create a triangular geometry const geometry = new v3d.Geometry(); geometry.vertices.push(new v3d.Vector3(-50, -50, 0)); geometry.vertices.push(new v3d.Vector3( 50, -50, 0)); geometry.vertices.push(new v3d.Vector3( 50, 50, 0)); //create a new face using vertices 0, 1, 2 const normal = new v3d.Vector3(0, 0, 1); //optional const color = new v3d.Color(0xffaa00); //optional const materialIndex = 0; //optional const face = new v3d.Face3(0, 1, 2, normal, color, materialIndex); //add the face to the geometry's faces array geometry.faces.push(face); //the face normals and vertex normals can be calculated automatically if not supplied above geometry.computeFaceNormals(); geometry.computeVertexNormals(); scene.add(new v3d.Mesh(geometry, material));

Конструктор

[name]([param:Integer a], [param:Integer b], [param:Integer c], [param:Vector3 normal], [param:Color color], [param:Integer materialIndex])

a — Индекс вершины A.
b — Индекс вершины B.
c — Индекс вершины C.

normal — (необязательно) Нормаль полигона ([page:Vector3 Vector3]) или массив вертексных нормалей. Если передан один вектор, то устанавливается [page:.normal], если массив из трех векторов, то устанавливается [page:.vertexNormals].

color — (необязательно) Цвет грани [page:Color color] или массив вершин [page:Color colors]. Если передан один вектор, то устанавливается [page:.color], если массив из трех векторов, то устанавливается [page:.vertexColors].

materialIndex — (необязательный) индекс массива материалов, который следует связать с полигоном.

Свойства

[property:Integer a]

Индекс вершины A.

[property:Integer b]

Индекс вершины B.

[property:Integer c]

Индекс вершины C.

[property:Vector3 normal]

Face normal - вектор, показывающий направление Face3. Если вычисляется автоматически (с помощью [page:Geometry.computeFaceNormals]), то это нормализованное поперечное произведение двух ребер треугольника. По умолчанию *(0, 0, 0)*.

[property:Color color]

Face color - для использования этого свойства, свойство [page:Material.vertexColors vertexColors] материала должно быть установлено в *true*.

[property:Array vertexNormals]

Массив из 3 [page:Vector3 vertex normals].

[property:Array vertexColors]

Массив из 3 вертексных цветов - для их использования свойство материала [page:Material.vertexColors vertexColors] должно быть установлено в *true*.

[property:Integer materialIndex]

Индекс материала (указывает на индекс в связанном массиве материалов). По умолчанию *0*.
