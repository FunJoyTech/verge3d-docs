[name]

Объект [page:Layers] назначает [page:Object3D] на 1 или более из 32 слоев, пронумерованных от 0 до 31 - внутренне слои хранятся как [link:https://en.wikipedia.org/wiki/Mask_(computing) bit mask], и по умолчанию все Object3D являются членами слоя 0.

Это можно использовать для управления видимостью - объект должен иметь общий слой с [page:Camera camera], чтобы быть видимым при рендеринге вида этой камеры.

Все классы, наследующие от [page:Object3D], имеют свойство [page:Object3D.layers], которое является экземпляром этого класса.

Примеры

[example:webgl_layers WebGL / layers]

Конструктор

[name]()

Создайте новый Layers object, членство в котором изначально установлено на слой 0.

Свойства

[property:Integer mask]

Битовая маска, хранящая информацию о том, к какому из 32 слоев в данный момент относится этот объект слоев.

Методы

[method:null disable]([param:Integer layer])

layer - целое число от 0 до 31.

Удалить членство этого *layer*.

[method:null enable]([param:Integer layer])

layer - целое число от 0 до 31.

Добавьте членство в этом *layer*.

[method:null set]([param:Integer layer])

layer - целое число от 0 до 31.

Установите членство в *layer* и удалите членство во всех остальных слоях.

[method:Boolean test]([param:Layers layers])

layers - Layers object

Возвращает true, если этот и переданный объект *layers* имеют хотя бы один общий слой.

[method:null toggle]([param:Integer layer])

layer - целое число от 0 до 31.

Переключение принадлежности к *слою*.

[method:null enableAll]()

Добавьте принадлежность ко всем слоям.

[method:null disableAll]()

Удалите членство со всех слоев.
