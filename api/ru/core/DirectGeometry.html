[name]

Этот класс используется для преобразования из [page:Geometry] в [page:BufferGeometry].

Конструктор

[name]()

При этом создается новый [name].

Свойства

[property:Integer id]

Уникальный номер для данного экземпляра [name].

[property:String name]

Необязательное имя. По умолчанию - пустая строка.

[property:Array type]

Строка 'DirectGeometry'.

[property:Array vertices]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Array normals]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Array colors]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Array uvs]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Array uvs2]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Array groups]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Array morphTargets]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Array skinWeights]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Array skinIndices]

Инициализируется как пустой массив, заполняется с помощью [page:.fromGeometry]().

[property:Box3 boundingBox]

Габаритная рамка для bufferGeometry, которая может быть рассчитана с помощью [page:.computeBoundingBox](). По умолчанию *null*.

[property:Sphere boundingSphere]

Габаритная сфера для bufferGeometry, которая может быть вычислена с помощью [page:.computeBoundingSphere](). По умолчанию *null*.

[property:Boolean verticesNeedUpdate]

По умолчанию false.

[property:Boolean normalsNeedUpdate]

По умолчанию false.

[property:Boolean colorsNeedUpdate]

По умолчанию false.

[property:Boolean uvsNeedUpdate]

По умолчанию false.

[property:Boolean groupsNeedUpdate]

По умолчанию false.

Методы

[property:null computeGroups]([param:Geometry geometry])

Вычислите части геометрии, которые имеют разные MaterialIndex. Смотрите [page:BufferGeometry.groups].

[property:null fromGeometry]([param:Geometry geometry])

Передайте экземпляр [page:Geometry] для преобразования.
