[page:Loader] →

[name]

Загрузчик для загрузки [page:Material] в формате JSON. Для загрузки файлов используется внутренний [page:FileLoader].

Пример Кода

// instantiate a loader const loader = new v3d.MaterialLoader(); // load a resource loader.load( // resource URL 'path/to/material.json', // onLoad callback function(material) { object.material = material; }, // onProgress callback function(xhr) { console.log((xhr.loaded / xhr.total * 100) + '% loaded'); }, // onError callback function(err) { console.log('An error happened'); } );

Конструктор

[name]([param:LoadingManager manager])

[page:LoadingManager manager] — [page:LoadingManager loadingManager] для использования загрузчиком. По умолчанию [page:LoadingManager v3d.DefaultLoadingManager].

Создает новый [name].

Свойства

Общие свойства см. в базовом классе [page:Loader].

[property:Object textures]

Объект, содержащий любые текстуры, используемые материалом. См. [page:.setTextures].

Методы

Общие методы см. в базовом классе [page:Loader].

[method:null load]([param:String url], [param:Function onLoad], [param:Function onProgress], [param:Function onError])

[page:String url] — путь или URL-адрес к файлу. Это также может быть [link:https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs Data URI].
[page:Function onLoad] — Будет вызван после завершения загрузки. Аргументом будет загруженный [page:Material].
[page:Function onProgress] — Вызывается во время выполнения загрузки. Аргументом будет событие прогресса.
[page:Function onError] — Вызывается при ошибках загрузки.

Начните загрузку из url.

[method:Material parse]([param:Object json])

[page:Object json] — Объект json, содержащий параметры материала.

Парсит структуру JSON и создает новый [page:Material] типа [page:String json.type] с параметрами, определенными в объекте json.

[method:MaterialLoader setTextures]([param:Object textures])

[page:Object textures] — объект, содержащий любые текстуры, используемые материалом.
