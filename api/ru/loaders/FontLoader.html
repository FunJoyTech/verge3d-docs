[name]

Класс для загрузки шрифта в формате JSON. Возвращает [page:Font Font], который представляет собой массив [page:Shape Shapes], представляющий шрифт. Для загрузки файлов используется внутренний [page:FileLoader].

Вы можете конвертировать шрифты онлайн с помощью [link:https://gero3.github.io/facetype.js/ facetype.js]

Пример Кода

const loader = new v3d.FontLoader(); const font = loader.load( // resource URL 'fonts/helvetiker_bold.typeface.json', // onLoad callback function(font) { // do something with the font scene.add(font); }, // onProgress callback function(xhr) { console.log((xhr.loaded / xhr.total * 100) + '% loaded'); }, // onError callback function(err) { console.log('An error happened'); } );

Конструктор

[name]([param:LoadingManager manager])

[page:LoadingManager manager] — [page:LoadingManager loadingManager] для использования загрузчиком. По умолчанию [page:LoadingManager v3d.DefaultLoadingManager].

Создает новоый [name].

Свойства

Общие свойства см. в базовом классе [page:Loader].

Методы

Общие методы см. в базовом классе [page:Loader].

[method:null load]([param:String url], [param:Function onLoad], [param:Function onProgress], [param:Function onError])

[page:String url] — путь или URL-адрес к файлу. Также может быть [link:https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs Data URI].
[page:Function onLoad] — Будет вызван после завершения загрузки. Аргументом будет загруженный [page:Font font].
[page:Function onProgress] — Вызывается в процессе загрузки. Аргументом будет экземпляр XMLHttpRequest, который содержит байты .[page:Integer total] и .[page:Integer loaded].
[page:Function onError] — Вызывается при ошибках загрузки.

Начните загрузку из url и передайте загруженную [page:Texture texture] в onLoad.

[method:Font parse]([param:Object json])

[page:Object json] — Структура JSON для парсинга.

Парсит структуру JSON и возвращает [page:Font].
