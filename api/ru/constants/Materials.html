Константы Материалов

Эти константы определяют свойства, общие для всех типов материалов, за исключением операций объединения текстур, которые применяются только к [page:MeshBasicMaterial.combine MeshBasicMaterial], [page:MeshLambertMaterial.combine MeshLambertMaterial] и [page:MeshPhongMaterial.combine MeshPhongMaterial].

Сторона

v3d.FrontSide v3d.BackSide v3d.DoubleSide

Определяет, с какой стороны будут отрисовываться полигоны - спереди, сзади или с обеих. По умолчанию [page:Constant FrontSide].

Режим смешивания

v3d.NoBlending v3d.NormalBlending v3d.AdditiveBlending v3d.SubtractiveBlending v3d.MultiplyBlending v3d.CustomBlending

Они управляют уравнениями смешивания источника и получателя для RGB и Alpha материала, отправляемыми в WebGLRenderer для использования WebGL.
[page:Constant NormalBlending] по умолчанию.
Обратите внимание, что для использования [page:Constant CustomBlending] необходимо установить [page:CustomBlendingEquation Custom Blending Equations].
Смотрите пример [example:webgl_materials_blending materials / blending].

Режим Глубины

v3d.NeverDepth v3d.AlwaysDepth v3d.LessDepth v3d.LessEqualDepth v3d.GreaterEqualDepth v3d.GreaterDepth v3d.NotEqualDepth

Функция глубины, которую материал использует для сравнения Z-глубины входящих пикселей с текущим значением Z-буфера глубины. Если результат сравнения true, пиксель будет отрисован.
[page:Materials NeverDepth] никогда не вернет true.
[page:Materials AlwaysDepth] всегда возвращает true..
[page:Materials LessDepth] вернет true, если Z-глубина входящего пикселя меньше Z-глубины текущего буфера.
[page:Materials LessEqualDepth] используется по умолчанию и возвращает true, если Z-глубина входящего пикселя меньше или равна Z-глубине текущего буфера.
[page:Materials GreaterEqualDepth] вернет true, если Z-глубина входящего пикселя больше или равна Z-глубине текущего буфера.
[page:Materials GreaterDepth] вернет true, если Z-глубина входящего пикселя больше, чем Z-глубина текущего буфера.
[page:Materials NotEqualDepth] вернет true, если Z-глубина входящего пикселя не равна Z-глубине текущего буфера.

Операции Объединения Текстур

v3d.MultiplyOperation v3d.MixOperation v3d.AddOperation

Они определяют, как результат цвета поверхности объединяется с картой окружения (если она присутствует), для [page:MeshBasicMaterial.combine MeshBasicMaterial], [page:MeshLambertMaterial.combine MeshLambertMaterial] and [page:MeshPhongMaterial.combine MeshPhongMaterial].
[page:Constant MultiplyOperation] используется по умолчанию и умножает цвет карты окружения на цвет поверхности.
[page:Constant MixOperation] использует отражательную способность для смешивания двух цветов.
[page:Constant AddOperation] добавляет два цвета.

Функции Стенсилов

v3d.NeverStencilFunc v3d.LessStencilFunc v3d.EqualStencilFunc v3d.LessEqualStencilFunc v3d.GreaterStencilFunc v3d.NotEqualStencilFunc v3d.GreaterEqualStencilFunc v3d.AlwaysStencilFunc

Какую функцию стенсила использует материал для определения необходимости выполнения операции стенсила.
[page:Materials NeverStencilFunc] никогда не вернет true.
[page:Materials LessStencilFunc] вернет true, если базовое значение стенсила меньше текущего значения стенсила.
[page:Materials EqualStencilFunc] вернет true, если базовое значение стенсила равно текущему значению стенсила.
[page:Materials LessEqualStencilFunc] вернет true, если базовое значение стенсила меньше или равно текущему значению стенсила.
[page:Materials GreaterStencilFunc] вернет true, если базовое значение стенсила больше, чем текущее значение стенсила.
[page:Materials NotEqualStencilFunc] вернет true, если базовое значение стенсила не равно текущему значению стенсила.
[page:Materials GreaterEqualStencilFunc] вернет true, если базовое значение стенсила больше или равно текущему значению стенсила.
[page:Materials AlwaysStencilFunc] всегда возвращает true.

Операции со Стенсилами

v3d.ZeroStencilOp v3d.KeepStencilOp v3d.ReplaceStencilOp v3d.IncrementStencilOp v3d.DecrementStencilOp v3d.IncrementWrapStencilOp v3d.DecrementWrapStencilOp v3d.InvertStencilOp

Какую операцию стенсила материал выполнит с пикселем буфера стенсила, если предоставленная функция стенсила пройдет.
[page:Materials ZeroStencilOp] установит значение стенсила равным 0.
[page:Materials KeepStencilOp] не изменит текущее значение стенсила.
[page:Materials ReplaceStencilOp] заменит значение стенсила на указанное опорное значение стенсила.
[page:Materials IncrementStencilOp] увеличит текущее значение стенсила на 1.
[page:Materials DecrementStencilOp] уменьшит текущее значение стенсила на 1.
[page:Materials IncrementWrapStencilOp] увеличит текущее значение стенсила на 1. Если значение увеличится после 255, оно будет установлено в 0.
[page:Materials DecrementWrapStencilOp] увеличит текущее значение стенсила на 1. Если значение уменьшится ниже 0, оно будет установлено на 255.
[page:Materials InvertStencilOp] выполнит побитовое преобразование текущего значения стенсила.
