[name]

Источник света, расположенный непосредственно над сценой, с переходом цвета от цвета неба к цвету земли.

Этот свет нельзя использовать для создания теней.

Пример Кода

const light = new v3d.HemisphereLight(0xffffbb, 0x080820, 1); scene.add(light);

Конструктор

[name]([param:Integer skyColor], [param:Integer groundColor], [param:Float intensity])

[page:Integer skyColor] - (необязательно) шестнадцатеричный цвет неба. По умолчанию 0xffffffff.
[page:Integer groundColor] - (необязательно) шестнадцатеричный цвет земли. По умолчанию 0xffffffff.
[page:Float intensity] - (необязательно) числовое значение strength/intensity света. По умолчанию 1.

Создает новый [name].

Свойства

Общие свойства смотрите в базовом классе [page:Light Light].

[property:Float color]

Цвет неба света, переданный в конструкторе. По умолчанию это новый [page:Color], установленный на белый (0xffffffff).

[property:Float groundColor]

Цвет основания(земли) света, переданный в конструкторе. По умолчанию это новый [page:Color], установленный на белый (0xffffffff).

[property:Vector3 position]

Он устанавливается равным [page:Object3D.DefaultUp] (0, 1, 0), чтобы свет светил сверху вниз.

Методы

Общие методы смотрите в базовом классе [page:Light Light].

[method:HemisphereLight copy]([param:HemisphereLight source])

Копирует значение [page:.color color], [page:.intensity intensity] и [page:.groundColor groundColor] из света [page:Light source] в этот свет.
