[name]

Этот класс содержит параметры, определяющие экспоненциальный квадратичный туман, который дает четкий вид вблизи камеры и быстрее экспоненциально сгущающийся туман дальше от камеры.

Конструктор

[name]([param:Integer color], [param:Float density])

Параметр color передается в конструктор [page:Color] для установки свойства color. Цвет может быть шестнадцатеричным целым числом или строкой в стиле CSS.

Свойства

[property:String name]

Необязательное имя объекта (необязательно уникальное). По умолчанию это пустая строка.

[property:Color color]

Цвет тумана. Пример: Если установить значение black, то дальние объекты будут рендериться черными.

[property:Float density]

Определяет, как быстро туман будет становиться плотным.

По умолчанию 0.00025.
