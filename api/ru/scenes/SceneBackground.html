[name]

Объект, представляющий настройки фона сцены. Предназначен для использования как свойство [page:Scene.background scene.background].

Конструктор

[name]([page:.data data])

Данные для заливки фона. Может быть задан [page:Color], который задает прозрачный цвет, [page:Texture], покрывающий canvas, [page:CubeTexture] или [page:WebGLRenderTargetCube].

Свойства

[property:null data]

Данные для заливки фона. Может быть задан [page:Color], который задает прозрачный цвет, [page:Texture], покрывающий canvas, [page:CubeTexture] или [page:WebGLRenderTargetCube].

[property:Boolean equirecInvertU]

Если установлено значение true, инвертирует отображение текстуры по координате U для [page:Texture Textures] с [page:Textures EquirectangularReflectionMapping]. По умолчанию установлено значение false.

[property:Number equirecOffsetU]

Определяет смещение в пространстве текстуры по координате U для [page:Texture Textures] с [page:Textures EquirectangularReflectionMapping]. По умолчанию равно 0.

[property:Boolean isSceneBackground]

Используется для проверки того, имеет ли данный или производные классы тип SceneBackground. По умолчанию - true.

Методы

[method:SceneBackground clone]()

Создает новый клон объекта SceneBackground.

[method:null copy]([page:SceneBackground source])

Копирует свойства из исходного SceneBackground в этот.

[method:null toJSON]()

Возвращает объектное представление объекта SceneBackground в формате JSON.
