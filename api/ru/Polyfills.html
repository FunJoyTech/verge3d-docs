Полифиллы

Verge3D включает полифиллы для следующих функций и констант.

[page:Number.EPSILON Number.EPSILON]

Разница между единицей и наименьшим значением, большим единицы, которое может быть представлено в виде числа. [link:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number/EPSILON MDN reference].

[page:Math.sign Math.sign]([page:Number x])

Если аргументом является положительное число, отрицательное число, положительный ноль или отрицательный ноль, функция возвращает 1, -1, 0 или -0 соответственно. В противном случае возвращается NaN. [link:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/sign MDN reference].

[page:Function.prototype.name Function.prototype.name]([page:Number x])

Возвращает имя функции или (до реализации ES6) пустую строку для анонимных функций. [link:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function/name MDN reference].

[page:Object.assign Object.assign]([page:Object target], [page:Object ...sources])

Метод Object.assign() используется для копирования значений всех перечислимых собственных свойств из одного или нескольких исходных объектов в целевой объект. Он возвращает целевой объект. [link:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/assign MDN reference].
