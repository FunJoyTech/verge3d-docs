[name]

Класс, содержащий полезные служебные функции для ваших 3D-приложений.

Методы

[method:Object getPageParams]

Разбирает GET-параметры страницы и возвращает объект с разобранными данными key-value.

[method:null loadScript](url, appendElem, callback)

url - URL загруженного скрипта.
appendElem - Элемент HTML для добавления загруженного сценария.
callback - обработчик события загрузки.
Загружает и выполняет заданный файл JavaScript.

[method:null drawWatermark](elem, text)

elem -- Объект, который нужно удалить из родителя.
text -- Сцена, на которую нужно прикрепить объект.
Добаляет водяной знак (копирайт и/или возможное информационное сообщение) к вашему приложению.
