[name]

Создает меш, который постоянно указывает на определенный HTML-элемент.

Конструктор

[name]([page:Color color], width)

Параметры:
  • color - цвет линии (в линейном пространстве), по умолчанию *(1, 0, 0)*..
  • width - ширина линии в пикселях, по умолчанию *2px*.

Свойства

Общие свойства смотрите в базовом классе [page:Mesh].

[property:HTMLElement elemHTML]

Объект HTMLElement для привязки строки. По умолчанию *null*.

[property:Boolean isLineHTML]

Используется для проверки того, является ли данный или производные классы мешами LineHTML. По умолчанию *true*.

[property:Integer offset]

Промежуток в пикселях между концом строки и целевым HTMLElement. По умолчанию *5px*.

[property:function onBeforeRender]

Обратный вызов, который выполняется непосредственно перед рендерингом LineHTML. Эта функция вызывается со следующими параметрами: рендерер, сцена, камера, геометрия, материал, группа.

По умолчанию этот обратный вызов обновляет геометрию линии на основе ее мировой позиции и позиции элемента HTMLElement, на который указывает линия.
