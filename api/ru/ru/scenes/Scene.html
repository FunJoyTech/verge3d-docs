[name]

Сцены позволяют установить, что и где будет рендерить Verge3D. Здесь вы размещаете объекты, освещение и камеры.

Конструктор

[name]()

Создаёт новый объект сцены.

Свойства

[property:Boolean autoUpdate]

По умолчанию равно true. Если установлено, то рендерер проверяет каждый кадр, нуждается ли сцена и ее объекты в обновлении матриц. Если это не так, то вы должны сами поддерживать все матрицы в сцене.

[property:Object background]

Если значение не равно null, задает фон, используемый при рендеринге сцены, и всегда рендерится первым. Может быть установлен в [page:Color], который задает прозрачный цвет, [page:Texture], покрывающий холст, кубическую карту как [page:CubeTexture] или [page:WebGLCubeRenderTarget] или равноугольную как [page:Texture]. По умолчанию - null.

[property:Texture environment]

Если значение не равно null, эта текстура устанавливается в качестве карты окружения для всех физических материалов в сцене. Однако невозможно перезаписать существующую текстуру, назначенную [page:MeshStandardMaterial.envMap]. По умолчанию - null.

[property:Fog fog]

Экземпляр [page:Fog fog], определяющий тип тумана, который влияет на все отрисованное в сцене. По умолчанию - null.

[property:Material overrideMaterial]

Если значение не равно null, это заставит все в сцене рендериться с этим материалом. По умолчанию - null.

[property:CubeReflectionProbe worldEnvMapProbe]

Экземпляр зонда отраженного света, который добавляет IBL окружения ко всем объектам сцены.

[property:Material worldMaterial]

Материал, используемый для создания мировой среды.
