[name]

Материал для рисования геометрии в виде wireframe с пунктирными линиями.

Пример Кода

const material = new v3d.LineDashedMaterial({ color: 0xffffff, linewidth: 1, scale: 1, dashSize: 3, gapSize: 1, });

Примеры

[example:webgl_lines_dashed WebGL / lines / dashed]

Конструктор

[name]([param:Object parameters])

[page:Object parameters] — (опционально) объект с одним или несколькими свойствами, определяющими внешний вид материала. Сюда можно передать любое свойство материала (включая любое свойство, унаследованное от [page:LineBasicMaterial]).

Свойства

Общие свойства см. в базовом классе [page:LineBasicMaterial].

[property:number dashSize]

Размер пунктира. Расстояние включает в себя и промежуток и штрих. По умолчанию *3*.

[property:number gapSize]

Размер промежутка. По умолчанию *1*.

[property:number scale]

Масштаб пунктирной части линии. По умолчанию *1*.

Методы

Общие методы смотрите в базовом классе [page:LineBasicMaterial].
