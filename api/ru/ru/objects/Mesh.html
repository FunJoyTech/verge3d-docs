[name]

Класс, представляющий триангулированные объекты [link:https://en.wikipedia.org/wiki/Polygon_mesh polygon mesh]. Также служит базой для других классов, таких как [page:SkinnedMesh]..

Пример Кода

const geometry = new v3d.BoxBufferGeometry(1, 1, 1); const material = new v3d.MeshBasicMaterial({ color: 0xffff00 }); const mesh = new v3d.Mesh(geometry, material); scene.add(mesh);

Конструктор

[name]([param:Geometry geometry], [param:Material material])

[page:Geometry geometry] — (необязательно) экземпляр [page:Geometry] или [page:BufferGeometry]. По умолчанию это новый [page:BufferGeometry].
[page:Material material] — (опциональный) один или массив [page:Material]. По умолчанию это новый [page:MeshBasicMaterial]

Свойства

Общие свойства см. в базовом классе [page:Object3D].

[property:Geometry geometry]

Экземпляр [page:Geometry] или [page:BufferGeometry] (или производных классов), определяющий структуру объекта.

Рекомендуется всегда использовать [page:BufferGeometry], если это возможно, для лучшей производительности.

[property:Material material]

Экземпляр материала, полученный из базового класса [page:Material] или массив материалов, определяющий внешний вид объекта. По умолчанию это [page:MeshBasicMaterial].

[property:Array morphTargetInfluences]

Массив весов, обычно от 0 до 1, который определяет, насколько сильно применяется морфинг. По умолчанию не определен, но сбрасывается в пустой массив командой [page:Mesh.updateMorphTargets updateMorphTargets].

[property:Object morphTargetDictionary]

Список morphTargets основанный на свойстве morphTarget.name. По умолчанию не определен, но перестраивается [page:Mesh.updateMorphTargets updateMorphTargets].

Методы

Общие методы см. в базовом классе [page:Object3D].

[method:Mesh clone]()

Возвращает клон данного объекта [name] и его преемников.

[method:null raycast]([param:Raycaster raycaster], [param:Array intersects])

Получить пересечения между отбрасываемым лучом и этим мешем. [page:Raycaster.intersectObject] вызовет этот метод, но результаты не будут упорядочены.

[method:null updateMorphTargets]()

Обновляет morphTargets так, чтобы они не оказывали влияния на объект. Сбрасывает свойства [page:Mesh.morphTargetInfluences morphTargetInfluences] и [page:Mesh.morphTargetDictionary morphTargetDictionary].
