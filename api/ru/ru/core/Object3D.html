[name]

Это базовый класс для большинства объектов в Verge3D, предоставляющий набор свойств и методов для манипулирования объектами в трехмерном пространстве.

Обратите внимание, что его можно использовать для группировки объектов с помощью метода [page:.add](object), который добавляет объект в качестве дочернего, однако для этого лучше использовать [page:Group].

Конструктор

[name]()

Конструктор не принимает никаких аргументов.

Свойства

[property:AnimationClip animations]

Массив с анимационными клипами объекта.

[property:Boolean castShadow]

Будет ли объект рендериться в карту теней. По умолчанию *false*.

[property:Object3D children]

Массив с дочерними объектами объекта. Информацию о ручном группировании объектов см. в [page:Group].

[property:Material customDepthMaterial]

Пользовательский материал глубины, который будет использоваться при рендеринге на карту глубины. Может использоваться только в контексте сетки. При создании теней с помощью [page:DirectionalLight] или [page:SpotLight], если вы (a) изменяете положение вершин в вершинном шейдере, (b) используете карту смещения, (c) используете альфа-карту с alphaTest или (d) используете прозрачную текстуру с alphaTest, вы должны указать пользовательский материал глубины (customDepthMaterial) для правильных теней. По умолчанию - *undefined*.

[property:Material customDistanceMaterial]

То же, что и [page:.customDepthMaterial customDepthMaterial], но используется с [page:PointLight]. По умолчанию - *undefined*.

[property:Boolean frustumCulled]

Если этот параметр установлен, каждый кадр проверяется, находится ли объект во фрустуме камеры, перед отрисовкой объекта. В противном случае объект будет отображаться каждый кадр, даже если он не виден. По умолчанию *true*.

[property:Array groupNames]

Массив, указывающий группы, к которым принадлежит данный объект. Используется для группировки/ссылки на коллекции объектов.

[property:Integer id]

readonly – Уникальный номер для данного экземпляра объекта.

[property:Layers layers]

Принадлежность объекта к слою. Объект виден, только если у него есть хотя бы один общий слой с используемой [page:Camera]. Это свойство также можно использовать для отсеивания нежелательных объектов в тестах пересечения лучей при использовании [page:Raycaster].

[property:Matrix4 matrix]

Матрица локального преобразования.

[property:Boolean matrixAutoUpdate]

Если этот параметр установлен, он вычисляет матрицу положения, (вращения или кватерниона) и масштаба каждый кадр, а также пересчитывает свойство matrixWorld. По умолчанию [page:Object3D.DefaultMatrixAutoUpdate] (true).

[property:Matrix4 matrixWorld]

Глобальное преобразование объекта. Если у объекта Object3D нет родителя, то оно идентично локальному преобразованию [page:.matrix].

[property:Boolean matrixWorldNeedsUpdate]

Когда это значение установлено, вычисляется matrixWorld в этом кадре и сбрасывается это свойство в false. По умолчанию *false*.

[property:Matrix4 modelViewMatrix]

Этот параметр передается в шейдер и используется для расчета положения объекта.

[property:String name]

Необязательное имя объекта (не обязательно уникальное). По умолчанию это пустая строка.

[property:Matrix3 normalMatrix]

Это значение передается в шейдер и используется для расчета освещения объекта. Это транспонирование обратной величины левой верхней подматрицы 3x3 матрицы modelViewMatrix этого объекта.

Причина создания этой специальной матрицы заключается в том, что простое использование матрицы modelViewMatrix может привести к неединичной длине нормалей (при масштабировании) или неперпендикулярному направлению (при неравномерном масштабировании).

С другой стороны, часть перевода матрицы modelViewMatrix не имеет значения для расчета нормалей. Поэтому достаточно матрицы Matrix3.

[property:Function onAfterRender]

Необязательный обратный вызов, который выполняется сразу после рендеринга Object3D. Эта функция вызывается со следующими параметрами: рендерер, сцена, камера, геометрия, материал, группа.

[property:Function onBeforeRender]

Необязательный обратный вызов, который выполняется непосредственно перед рендерингом Object3D. Эта функция вызывается со следующими параметрами: рендерер, сцена, камера, геометрия, материал, группа.

[property:Object3D parent]

Родитель объекта в [link:https://en.wikipedia.org/wiki/Scene_graph scene graph]. У объекта может быть не более одного родителя.

[property:Vector3 position]

[page:Vector3], представляющий локальную позицию объекта. По умолчанию (0, 0, 0)..

[property:Quaternion quaternion]

Локальное вращение объекта как [page:Quaternion Quaternion]..

[property:Boolean receiveShadow]

Принимает ли материал тени. По умолчанию *false*..

[property:Number renderOrder]

Это значение позволяет переопределить порядок рендеринга объектов [link:https://en.wikipedia.org/wiki/Scene_graph scene graph] по умолчанию, хотя непрозрачные и прозрачные объекты остаются отсортированными независимо. Когда это свойство установлено для экземпляра [page:Group Group], все объекты-потомки будут отсортированы и отрисованы вместе. Сортировка производится от низшего к высшему порядку рендеринга. Значение по умолчанию - *0*.

[property:Euler rotation]

Локальное вращение объекта (см. [link:https://en.wikipedia.org/wiki/Euler_angles Euler angles]), в радианах.

[property:Vector3 scale]

Локальный масштаб объекта. По умолчанию [page:Vector3](1, 1, 1).

[property:Vector3 up]

Это используется методом [page:.lookAt lookAt], например, для определения ориентации результата.
По умолчанию это [page:Object3D.DefaultUp] — то есть (0, 1, 0).

[property:Object userData]

Объект, который можно использовать для хранения пользовательских данных об объекте Object3D. Он не должен содержать ссылки на функции, так как они не будут клонироваться.

[property:String uuid]

[link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID] данного экземпляра объекта. Он присваивается автоматически, поэтому его не следует редактировать.

[property:Boolean visible]

Объект отображается, если *true*. По умолчанию *true*.

Статические Свойства

Статические свойства и методы определяются для каждого класса, а не для каждого экземпляра этого класса. Это означает, что изменение [page:Object3D.DefaultUp] или [page:Object3D.DefaultMatrixAutoUpdate] изменит значения [page:.up up] и [page:.matrixAutoUpdate matrixAutoUpdate] для каждого экземпляра Object3D (или производных классов), созданный после внесения изменения (уже созданные Object3D не будут затронуты).

[property:Vector3 DefaultUp]

Направление по умолчанию [page:.up up] для объектов, также используется как положение по умолчанию для [page:DirectionalLight], [page:HemisphereLight] и [page:Spotlight] (которые создают свет, светящий сверху вниз).
По умолчанию имеет значение (0, 1, 0).

[property:Boolean DefaultMatrixAutoUpdate]

Настройка по умолчанию для [page:.matrixAutoUpdate matrixAutoUpdate] для вновь созданных Object3D.

Методы

[page:EventDispatcher EventDispatcher] methods are available on this class.

[method:this add]([param:Object3D object], ...)

Добавляет *object* в качестве дочернего объекта данного объекта. Может быть добавлено произвольное количество объектов. Любой текущий родитель объекта, переданного здесь, будет удален, так как объект может иметь не более одного родителя.

Информацию о ручном группировании объектов см. в [page:Group].

[method:null applyMatrix4]([param:Matrix4 matrix])

Применяет матричное преобразование к объекту и обновляет положение, поворот и масштаб объекта.

[method:this applyQuaternion]([param:Quaternion quaternion])

Применяет к объекту вращение, представленное кватернионом.

[method:this attach]([param:Object3D object])

Добавляет *объект* в качестве дочернего объекта, сохраняя преобразование мира объекта.

[method:Object3D clone]([param:Boolean recursive])

recursive -- если true, потомки объекта также клонируются. По умолчанию - true.

Возвращает клон этого объекта и, по желанию, всех потомков.

[method:this copy]([param:Object3D object], [param:Boolean recursive])

recursive -- если true, потомки объекта также копируются. По умолчанию - true.

Копирует заданный объект в этот объект. Примечание: слушатели событий и пользовательские обратные вызовы ([page:.onAfterRender] и [page:.onBeforeRender]) не копируются.

[method:Object3D getObjectById]([param:Integer id])

id -- Уникальный номер экземпляра объекта

Выполняет поиск объекта и его дочерних элементов, начиная с самого объекта, и возвращает первый с совпадающим id.
Обратите внимание, что идентификаторы присваиваются в хронологическом порядке: 1, 2, 3, ..., увеличиваясь на единицу для каждого нового объекта.

[method:Object3D getObjectByName]([param:String name])

name -- Строка для сопоставления с дочерним свойством Object3D.name.

Выполняет поиск объекта и его дочерних элементов, начиная с самого объекта, и возвращает первый с подходящим именем.
Обратите внимание, что для большинства объектов имя по умолчанию является пустой строкой. Вам придется задать его вручную, чтобы воспользоваться этим методом.

[method:Object3D getObjectByProperty]([param:String name], [param:Float value])

name -- имя свойства для поиска.
value -- значение данного свойства.

Выполняет поиск по объекту и его дочерним объектам, начиная с самого объекта, и возвращает первый объект, свойство которого соответствует заданному значению.

[method:Vector3 getWorldPosition]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает вектор, представляющий положение объекта в мировом пространстве.

[method:Quaternion getWorldQuaternion]([param:Quaternion target])

[page:Quaternion target] — результат будет скопирован в этот кватернион.

Возвращает кватернион, представляющий вращение объекта в мировом пространстве.

[method:Vector3 getWorldScale]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает вектор масштабных коэффициентов, применяемых к объекту для каждой оси в мировом пространстве.

[method:Vector3 getWorldDirection]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает вектор, представляющий направление положительной оси z объекта в мировом пространстве.

[method:Vector3 localToWorld]([param:Vector3 vector])

vector - Вектор, представляющий позицию в локальном пространстве этого объекта.

Преобразует вектор из локального пространства этого объекта в мировое пространство.

[method:null lookAt]([param:Vector3 vector])
[method:null lookAt]([param:Float x], [param:Float y], [param:Float z])

vector - Вектор, представляющий позицию в мировом пространстве.

Опционально, компоненты [page:.x x], [page:.y y] и [page:.z z] положения в мировом пространстве..

Поворачивает объект лицом к точке в мировом пространстве.

Этот метод не поддерживает объекты с неоднородно масштабированными родителями.

[method:Array raycast]([param:Raycaster raycaster], [param:Array intersects])

Абстрактный (пустой) метод для получения пересечений между брошенным лучом и данным объектом. Подклассы, такие как [page:Mesh], [page:Line] и [page:Points], реализуют этот метод, чтобы использовать отбрасывание лучей.

[method:this remove]([param:Object3D object], ...)

Удаляет *объект* как дочерний объект данного объекта. Может быть удалено произвольное количество объектов.

[method:this clear]()

Удаляет все дочерние объекты.

[method:this rotateOnAxis]([param:Vector3 axis], [param:Float angle])

axis -- Нормированный вектор в пространстве объектов.
angle -- Угол в радианах.

Поворот объекта вдоль оси в пространстве объекта. Предполагается, что ось нормализована.

[method:this rotateOnWorldAxis]([param:Vector3 axis], [param:Float angle])

axis -- Нормированный вектор в мировом пространстве.
angle -- Угол в радианах.

Поворот объекта вдоль оси в мировом пространстве. Предполагается, что ось нормализована. Метод Предполагает отсутствие повернутого родителя.

[method:this rotateX]([param:Float rad])

rad - угол поворота в радианах.

Поворачивает объект вокруг оси x в локальном пространстве.

[method:this rotateY]([param:Float rad])

rad - угол поворота в радианах.

Поворачивает объект вокруг оси y в локальном пространстве.

[method:this rotateZ]([param:Float rad])

rad - угол поворота в радианах.

Поворачивает объект вокруг оси z в локальном пространстве.

[method:null setRotationFromAxisAngle]([param:Vector3 axis], [param:Float angle])

axis -- Нормированный вектор в пространстве объектов.
angle -- угол в радианах

Вызывает [page:Quaternion.setFromAxisAngle setFromAxisAngle]([page:Float axis], [page:Float angle]) на [page:.quaternion].

[method:null setRotationFromEuler]([param:Euler euler])

euler -- Угол Эйлера, определяющий величину поворота.
Вызывает [page:Quaternion.setRotationFromEuler setRotationFromEuler]([page:Euler euler]) на [страница:.кватернион].

[method:null setRotationFromMatrix]([param:Matrix4 m])

m -- повернуть кватернион по компоненту вращения матрицы.
Вызывает [page:Quaternion.setFromRotationMatrix setFromRotationMatrix]([page:Matrix4 m]) на [page:.quaternion].

Обратите внимание, что это предполагает, что верхние 3x3 из m являются чистой матрицей вращения (т.е. немасштабированной).

[method:null setRotationFromQuaternion]([param:Quaternion q])

q -- нормализованный кватернион.

Скопируйте заданный кватернион в [page:.quaternion].

[method:this translateOnAxis]([param:Vector3 axis], [param:Float distance])

axis -- Нормализованный вектор в пространстве объекта.
distance -- Расстояние для перевода.

Перевести объект на расстояние вдоль оси в пространстве объекта. Предполагается, что ось нормализована.

[method:this translateX]([param:Float distance])

Перемещает объект вдоль оси x в пространстве объекта на единицы *distance*.

[method:this translateY]([param:Float distance])

Переводит объект вдоль оси y в пространстве объекта на единицы *distance*.

[method:this translateZ]([param:Float distance])

Перемещает объект вдоль оси z в пространстве объекта на единицы *distance*.

[method:null traverse]([param:Function callback])

callback - Функция, первым аргументом которой является объект object3D.

Выполняет обратный вызов для этого объекта и всех потомков.
Примечание: Не рекомендуется изменять граф сцены внутри обратного вызова.

[method:null traverseVisible]([param:Function callback])

callback - Функция, первым аргументом которой является объект object3D.

Аналогично traverse, но обратный вызов будет выполняться только для видимых объектов. Потомки невидимых объектов не обходятся.
Примечание: Не рекомендуется изменять граф сцены внутри обратного вызова.

[method:null traverseAncestors]([param:Function callback])

callback - Функция, первым аргументом которой является объект object3D.

Выполняет обратный вызов для всех предков.
Примечание: Не рекомендуется изменять граф сцены внутри обратного вызова.

[method:null updateMatrix]()

Обновляет локальное преобразование.

[method:null updateMatrixWorld]([param:Boolean force])

Обновляет глобальное преобразование объекта и его потомков.

[method:null updateWorldMatrix]([param:Boolean updateParents], [param:Boolean updateChildren])

updateParents - рекурсивно обновляет глобальное преобразование предков.
updateChildren - рекурсивно обновляет глобальное преобразование потомков.

Обновляет глобальное преобразование объекта.

[method:Vector3 worldToLocal]([param:Vector3 vector])

vector - Вектор, представляющий положение в мировом пространстве.

Преобразует вектор из мирового пространства в локальное пространство этого объекта.
