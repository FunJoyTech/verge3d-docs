[name]

Класс, содержащий полезные методы рендеринга.

Методы

[method:WebGLCubeRenderTarget renderSceneToCubemap]([param:Scene scene], [param:WebGLRenderer renderer], [param:WebGLCubeRenderTarget cubeRenderTarget], [param:Object cameraOptions])

Рендеринг заданной сцены в cubemap. Параметры:
  • scene - [page:Scene] для рендеринга в cubemap
  • renderer - [page:WebGLRenderer]
  • cubeRenderTarget - [page:WebGLCubeRenderTarget] render target содержащая cubemap для рендеринга
  • cameraOptions - объект, содержащий параметры для кубической камеры, используемой для рендеринга сцены в cubemap; параметры и их значения по умолчанию: { camPos: new Vector3(), clipStart: 0.1, clipEnd: 100, } camPos - позиция кубической камеры
    clipStart - значение ближней планки
    clipEnd - значение дальней планки

[method:WebGLCubeRenderTarget renderWorldNodeMatToCubemap]([param:WebGLRenderer renderer], [param:MeshNodeMaterial material], [param:Integer cubeRes], [param:Object options])

Рендеринг материала ноды мира в cubemap. Параметры:
  • renderer - [page:WebGLRenderer]
  • material - материал ноды мира, должен иметь тип [page:MeshNodeMaterial]
  • cubeRes - разрешение отрендеренной cubemap
  • options - (необязательный) объект, содержащий параметры текстуры для автоматически генерируемой cube render target; имеет тот же формат как параметр *options* для конструктора[page:WebGLCubeRenderTarget]
Полученный [page:WebGLCubeRenderTarget render target] с текстурой куба можно использовать в качестве фона для сцены.
