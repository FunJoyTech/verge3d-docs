[name]

Здесь хранится ссылка на реальное свойство в графе сцены; используется внутренне.

Конструктор

[name]([param:Object3D rootNode], path, parsedPath)

-- [page:Object3D rootNode]: -- path -- parsedPath (optional)

Свойства

[property:Number path]

[property:Number parsedPath]

[property:Number node]

[property:Number rootNode]

[property:Object BindingType]

[property:Object Versioning]

[property:Array GetterByBindingType]

[property:Array SetterByBindingTypeAndVersioning]

Методы

[method:null getValue]([param:Array targetArray], [param:Number offset])

[method:null setValue]([param:Array sourceArray], [param:Number offset])

[method:null bind]()

Создает пару getter / setter для свойства в графе сцены. Используется внутри [page:PropertyBinding.getValue getValue] и [page:PropertyBinding.setValue setValue].

[method:null unbind]()

Отвязка пары getter / setter для свойства в графе сцены.

[method:Constructor Composite](targetGroup, path, optionalParsedPath)

Создайте новую составную привязку свойств (Composite PropertyBinding).

[method:Constructor create](root, path, parsedPath)

Создайте новую Composite PropertyBinding (если корнем является [page:AnimationObjectGroup]) или PropertyBinding.

[method:Constructor parseTrackName](trackName)

Сопоставляет строки в следующих формах:
-- nodeName.property
-- nodeName.property[accessor]
-- nodeName.material.property[accessor]
-- uuid.property[accessor]
-- uuid.objectName[objectIndex].propertyName[propertyIndex]
-- parentName/nodeName.property
-- parentName/parentName/nodeName.property[index]
-- .bone[Armature.DEF_cog].position
-- scene:helium_balloon_model:helium_balloon_model.position

[method:Constructor findNode](root, nodeName)

Найдите узел в дереве узлов или [page:Skeleton Skeleton].
