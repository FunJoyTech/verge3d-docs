[name]

Класс для генерации геометрии додекаэдра.

Конструктор

[name]([param:Float radius], [param:Integer detail])

radius — Радиус додекаэдра. По умолчанию равен 1.
detail — По умолчанию 0. Установка значения больше 0 добавляет вершины, в результате чего додекаэдр перестает быть додекаэдром.

Свойства

Общие свойства см. в базовом классе [page:Geometry].

[property:Object parameters]

Объект со свойством для каждого из параметров конструктора. Любая модификация после построения не изменяет геометрию.
