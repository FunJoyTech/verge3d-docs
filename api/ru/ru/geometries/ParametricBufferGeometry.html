[page:BufferGeometry] →

[name]

Генерирует геометрию, представляющую параметрическую поверхность.

Пример Кода

const geometry = new v3d.ParametricBufferGeometry(v3d.ParametricGeometries.klein, 25, 25); const material = new v3d.MeshBasicMaterial({ color: 0x00ff00 }); const klein = new v3d.Mesh(geometry, material); scene.add(klein);

Конструктор

[name]([param:Function func], [param:Integer slices], [param:Integer stacks])

func — Функция, которая принимает значения [page:Float u] и [page:Float v] от 0 до 1 и изменяет третий аргумент [page:Vector3]
slices — Количество срезов, используемых для параметрической функции
stacks — Количество стеков, используемых для параметрической функции

Свойства

Общие свойства см. в базовом классе [page:BufferGeometry].

[property:Object parameters]

Объект со свойством для каждого из параметров конструктора. Любая модификация после построения не изменяет геометрию.
