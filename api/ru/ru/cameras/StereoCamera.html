Двойные [page:PerspectiveCamera PerspectiveCamera] используются для таких эффектов, как [link:https://en.wikipedia.org/wiki/Anaglyph_3D 3D Anaglyph] или [link:https://en.wikipedia.org/wiki/parallax_barrier Parallax Barrier].

Примеры

[example:webgl_effects_anaglyph effects / anaglyph]
[example:webgl_effects_parallaxbarrier effects / parallaxbarrier]
[example:webgl_effects_stereo effects / stereo]

Конструктор

[name]()

Свойства

[property:Float aspect]

По умолчанию *1*.

[property:Float eyeSep]

По умолчанию *0.064*.

[property:PerspectiveCamera cameraL]

Левая камера. Добавляется в [page:Layers layer 1] — объекты, которые будут отображаться левой камерой, также должны быть добавлены в этот слой.

[property:PerspectiveCamera cameraR]

Правая камера. Она добавляется в [page:Layers layer 2] — объекты, которые будут отображаться правой камерой, также должны быть добавлены в этот слой.

Методы

[method:null update]([param:PerspectiveCamera camera])

Обновление стереокамер на основе переданной камеры.
