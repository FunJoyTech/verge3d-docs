[name]

Камера, использующая [link:https://en.wikipedia.org/wiki/Orthographic_projection orthographic projection].

В этом режиме проекции размер объекта на отрендеренном изображении остается постоянным независимо от расстояния до камеры.

Это может быть полезно, в частности, для рендеринга 2D-сцен и элементов пользовательского интерфейса.

Пример Кода

const camera = new v3d.OrthographicCamera(width / - 2, width / 2, height / 2, height / - 2, 1, 1000); scene.add(camera);

Конструктор

[name]([param:Number left], [param:Number right], [param:Number top], [param:Number bottom], [param:Number near], [param:Number far])

left — Левая плоскость фрустума камеры.
right — Правая плоскость фрустума камеры.
top — Верхняя плоскость фрустума камеры.
bottom — Нижняя плоскость фрустума камеры.
near — Ближняя плоскость фрустума камеры.
far — Дальняя плоскость фрустума камеры.

Вместе они определяют [link:https://en.wikipedia.org/wiki/Viewing_frustum viewing frustum].

Свойства

Общие свойства смотрите в базовом классе [page:Camera].
Обратите внимание, чтобы изменения, в большинстве из этих свойств, вступили в силу после внесения, вам нужно будет вызвать [page:OrthographicCamera.updateProjectionMatrix .updateProjectionMatrix],

[property:Float bottom]

Нижняя плоскость фрустума камеры.

[property:Float far]

Дальняя плоскость фрустума камеры. По умолчанию *2000*.

Должно быть больше, чем текущее значение плоскости [page:.near near].

[property:Float left]

Левая плоскость фрустума камеры.

[property:Float near]

Ближняя плоскость фрустума камеры. По умолчанию *0.1*.

Диапазон допустимых значений - от 0 до текущего значения плоскости [page:.far far]. Обратите внимание, что, в отличие от [page:PerspectiveCamera], *0* является допустимым значением для ближней плоскости ортографической камеры.

[property:Float right]

Правая плоскость фрустума камеры.

[property:Float top]

Верхнаяя плоскость фрустума камеры.

[property:Object view]

Устанавливается [page:OrthographicCamera.setViewOffset setViewOffset]. По умолчанию *null*.

[property:number zoom]

Получает или устанавливает коэффициент масштабирования камеры. По умолчанию *1*.

Методы

Общие методы смотрите в базовом классе [page:Camera].

[method:null setViewOffset]([param:Float fullWidth], [param:Float fullHeight], [param:Float x], [param:Float y], [param:Float width], [param:Float height])

fullWidth — полная ширина многоракурсной установки
fullHeight — полная высота многоракурсной установки
x — горизонтальное смещение подкамеры
y — вертикальное смещение подкамеры
width — ширина подкамеры
height — высота подкамеры

Устанавливает смещение в большем [link:https://en.wikipedia.org/wiki/Viewing_frustum viewing frustum]. Это полезно для многооконных или многомониторных настроек. Пример использования смотрите в [page:PerspectiveCamera.setViewOffset PerspectiveCamera].

[method:null clearViewOffset]()

Удаляет любое смещение, установленное методом .setViewOffset.

[method:null updateProjectionMatrix]()

Обновляет матрицу проекции камеры. Должно вызываться после любого изменения параметров.
