[name]

Этот класс работает так же, как [page:ShaderMaterial], за исключением того, что определения встроенных форм и атрибутов не добавляются автоматически в код шейдера GLSL.

Пример Кода

const material = new v3d.RawShaderMaterial({ uniforms: { time: { value: 1.0 } }, vertexShader: document.getElementById('vertexShader').textContent, fragmentShader: document.getElementById('fragmentShader').textContent, });

Конструктор

[name]([param:Object parameters])

[page:Object parameters] - (опционально) объект с одним или несколькими свойствами, определяющими внешний вид материала. Сюда можно передать любое свойство материала (включая любые свойства, унаследованные от [page:Material] и [page:ShaderMaterial]).
