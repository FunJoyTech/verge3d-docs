[name]

Специальный материал для рисования линий. Предназначен для использования с мешами [page:MeshLine] и [page:MeshLineIndexed].

Конструктор

[name]([page:Material parameters])

[page:Material parameters] - (необязательно) объект с одним или несколькими свойствами, определяющими внешний вид материала. Сюда можно передать любое свойство материала (включая любое свойство, унаследованное от [page:Material]).

Свойства

Общие свойства смотрите в базовом классе [page:Material].

[property:Texture alphaMap]

Текстура, которая управляет непрозрачностью вдоль линии (требуется, чтобы [page:.useAlphaMap useAlphaMap] имел значение true).

[property:Number alphaTest]

Пороговое альфа-значение для отбрасывания (от 0 до 1).

[property:Color color]

Цвет линии.

[property:Boolean isMeshLineMaterial]

Используется для проверки того, является ли данный или производные классы материалами линии меша. По умолчанию - true. Не следует изменять это значение, так как оно используется для внутренней оптимизации.

[property:Number lineWidth]

Ширина линии (если [page:.sizeAttenuation sizeAttenuation] равен true, то это мировые единицы; в противном случае - пиксели экрана).

[property:Texture map]

Текстура, управляющая цветом вдоль линии (требуется, чтобы [page:.useMap useMap] имел значение true).

[property:Number opacity]

Альфа-значение линии (от 0 до 1).p useMap] установлено в true).

[property:Vector2 repeat]

Вектор для масштабирования UV-координат [page:.map map] и [page:.alphaMap alphaMap].

[property:Vector2 resolution]

Вектор, определяющий размер холста.

[property:Number sizeAttenuation]

Сделать ширину линии постоянной (если 0) или нет (если 1).

[property:Number useAlphaMap]

Используйте текстуру [page:.alphaMap alphaMap].

[property:Number useMap]

Используйте текстуру [page:.map map].

[property:Number visibility]

Управление видимостью частиц вдоль линии (от 0 до 1) путем изменения ее альфа-значения.
