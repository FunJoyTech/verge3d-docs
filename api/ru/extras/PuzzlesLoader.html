[name]

Класс, способный загружать редактор пазлов и код визуальной логики.

Конструктор

[name]()

Свойства

[property:String editorCSSFile]

Устаревшее. Путь к CSS-файлу редактора Пазлов относительно исходной папки Пазлов (обычно это папка "puzzles/" SDK). По умолчанию это "puzzles.css".

[property:String editorJSFile]

Устаревшее. Путь к JS-файлу редактора Пазлов относительно исходного каталога Пазлов (обычно это папка "puzzles/" SDK). По умолчанию это "puzzles.min.js".

[property:String editorMediaDir]

Устаревшее. Путь к папке media редактора Puzzles относительно исходного каталога Puzzles (обычно это папка "puzzles/" SDK). По умолчанию - "media/".

[property:String libraryXML]

Устаревшее. Путь к файлу библиотеки редактора Puzzles относительно исходного каталога Puzzles (обычно это папка "puzzles/" SDK). По умолчанию это "library.xml".

Методы

[method:null loadEditorWithLogic](editorDir, logicURL, onLoad, onError)

Загрузка редактора Пазлов и файла логики. Методу необходимо указать путь к исходной папке редактора Пазлов (задается параметром editorDir, обычно это папка "puzzles/" SDK) и путь к JS-файлу логики (параметр logicURL). Обратный вызов onLoad можно использовать для загрузки сцены после того, как редактор и файл логики были загружены.

[method:null loadLogic](logicURL, onLoad, onError)

Загружает файл логики. Обратный вызов onLoad можно использовать для загрузки сцены после загрузки файла логики.
