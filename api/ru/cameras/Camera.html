[name]

Абстрактный базовый класс для камер. Этот класс всегда должен быть унаследован при создании новой камеры.

Конструктор

[name]()

Создает новый [name]. Обратите внимание, что этот класс не предназначен для прямого вызова; вероятно, вместо него вам понадобится [page:PerspectiveCamera] или [page:OrthographicCamera].

Совйства

Общие свойства смотрите в базовом классе [page:Object3D].

[property:Layers layers]

[page:Layers layers], членом которого является камера. Это наследуемое свойство от [page:Object3D].

Объекты должны иметь хотя бы один общий слой с камерой, чтобы быть видимыми при визуализации точки зрения камеры.

[property:Matrix4 matrixWorldInverse]

Это обратная величина по отношению к matrixWorld. MatrixWorld содержит матрицу, которая имеет мировое преобразование камеры.

[property:Matrix4 projectionMatrix]

Это матрица, которая содержит проекцию.

[property:Matrix4 projectionMatrixInverse]

Обратная проекция матрицы projectionMatrix.

Методы

Общие методы смотрите в базовом классе [page:Object3D].

[method:Camera clone]()

Возвращает новую камеру с теми же свойствами, что и эта.

[method:Camera copy]([param:Camera source], [param:Boolean recursive])

Копирует свойства из исходной камеры в эту.

[method:Vector3 getWorldDirection]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает [page:Vector3], представляющий направление в мировом пространстве, в котором смотрит камера. (Примечание: камера смотрит вниз по своей локальной, отрицательной оси z).
