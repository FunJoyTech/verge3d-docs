[name]

A class containing useful rendering methods.

Methods

[method:WebGLRenderTargetCube renderWorldNodeMatToCubemap]([page:WebGLRenderer renderer], [page:MeshNodeMaterial material], [page:Number cubeRes])

Render a world node material to a cubemap. The parameters are:
  • renderer - [page:WebGLRenderer]
  • material - world node material, should be of type [page:MeshNodeMaterial]
  • cubeRes - resolution of the rendered cubemap
The resulting [page:WebGLRenderTargetCube render target] with a cube texture in it can be used as a scene background.
