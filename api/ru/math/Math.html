[method:Float smoothstep]([param:Float x], [param:Float min], [param:Float max])

[page:Float x] - Значение для оценки на основе его положения между min и max.
[page:Float min] - Любое значение x ниже min будет равно 0.
[page:Float max] - Любое значение x, превышающее max, будет равно 1.

Возвращает значение от 0 до 1, которое представляет процент того, что x переместился между min и max, но сглаживается или замедляется, чем ближе X к min и max.

Ознакомьтесь для дополнительной информации [link:http://en.wikipedia.org/wiki/Smoothstep Smoothstep].

[method:Float smootherstep]([param:Float x], [param:Float min], [param:Float max])

[page:Float x] - Значение для оценки на основе его положения между min и max.
[page:Float min] - Любое значение x ниже min будет равно 0.
[page:Float max] - Любое значение x, превышающее max, будет равно 1.

Возвращает значение в диапазоне 0-1. Вариация функции smoothstep [link:https://en.wikipedia.org/wiki/Smoothstep#Variations variation on smoothstep], которая имеет нулевые производные первого и второго порядка при x=0 и x=1.
