[name]

Свойство буферизованного графа сцены, позволяющее взвешенное накопление; используется внутренне.

Конструктор

[name]([param:PropertyBinding binding], [param:String typeName], [param:Number valueSize])

-- binding
-- typeName
-- valueSize

Свойства

[property:PropertyBinding binding]

[property:TypedArray buffer]

Буфер с размером [page:PropertyMixer valueSize] * 4.

Имеет следующий вид: [incoming | accu0 | accu1 | orig]

Интерполяторы могут использовать .buffer в качестве своего .result, после чего данные поступают в 'incoming'. 'accu0' и 'accu1' используются с чередованием кадров для суммарного результата и сравниваются для обнаружения изменений. 'orig' хранит исходное состояние свойства.

[property:Number cumulativeWeight]

По умолчанию *0*.

[property:Number valueSize]

[property:Number referenceCount]

По умолчанию *0*.

[property:Number useCount]

По умолчанию *0*.

Methods

[method:null accumulate]([param:Number accuIndex], [param:Number weight])

Накопление данных в [page:PropertyMixer.buffer buffer][accuIndex] 'incoming' области в 'accu[i]'.
Если вес равен *0*, это ничего не дает.

[method:null apply]([param:Number accuIndex])

Применение состояния [page:PropertyMixer.buffer buffer] 'accu[i]' к привязке, когда accus отличается.

[method:null saveOriginalState]()

Запоминает состояние связанного свойства и копирует его в оба accus.

[method:null restoreOriginalState]()

Применяет к привязке состояние, принятое ранее через 'saveOriginalState'.
