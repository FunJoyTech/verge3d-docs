Пример Кода

const geometry = new v3d.PlaneBufferGeometry(5, 20, 32); const material = new v3d.MeshBasicMaterial({color: 0xffff00, side: v3d.DoubleSide}); const plane = new v3d.Mesh(geometry, material); scene.add(plane);

Конструктор

[name]([param:Float width], [param:Float height], [param:Integer widthSegments], [param:Integer heightSegments])

width — Ширина вдоль оси X. По умолчанию 1.
height — Высота вдоль оси Y. По умолчанию 1.
widthSegments — Опционально. По умолчанию 1.
heightSegments — Опционально. По умолчанию 1.
