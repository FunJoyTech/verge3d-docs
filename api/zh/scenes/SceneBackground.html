[name]

表示场景背景设置的对象。用作场景的[page:Scene.background scene.background]属性。

构造函数

[name]([page:.data data])

用于填充背景的数据。可以设置一个颜色设置清晰的[page:Color]颜色，一个[page:Texture]纹理覆盖画布，一个[page:CubeTexture]立方体纹理或一个[page:WebGLRenderTargetCube]。

属性

[property:null data]

用于填充背景的数据。可以设置一个颜色设置清晰的[page:Color]颜色，一个[page:Texture]纹理覆盖画布，一个[page:CubeTexture]立方体纹理或一个[page:WebGLRenderTargetCube]。

[property:Boolean equirecInvertU]

如果设置为true，沿 [page:Texture Textures]纹理的U坐标与[page:Textures EquirectangularReflectionMapping]反转纹理映射。默认是false。

[property:Number equirecOffsetU]

使用[page:Textures EquirectangularReflectionMapping]在纹理空间中沿[page:Texture Textures]纹理U坐标定义一个偏移量。默认值为0。

[property:Boolean isSceneBackground]

用于检查此类或派生类是否为SceneBackground类型。默认是true。

方法

[method:SceneBackground clone]()

创建SceneBackground对象的新克隆。

[method:null copy]([page:SceneBackground source])

将源 SceneBackground 中的属性复制到该属性中。

[method:null toJSON]()

返回 SceneBackground 的 JSON 对象格式。
