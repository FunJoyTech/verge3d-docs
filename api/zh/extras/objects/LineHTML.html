[page:Mesh] →

[name]

创建一个线条网格物体，该物体始终指向指定的 HTML 元素。

构造函数

[name]([page:Color color], width)

参数：
  • color - 线的颜色（在线性空间中），默认为 *(1, 0, 0)*。
  • width - 线的宽度（以像素为单位），默认为 *2px*.

属性

通用属性查看 [page:Mesh] 基类。

[property:HTMLElement elemHTML]

将要绑定到的 HTML元素对象。默认是 *null*.

[property:Boolean isLineHTML]

用于检查该类或派生类是否为 LineHTML 网格。默认是 *true*。

[property:Integer offset]

线的末端和目标 HTML元素之间的间距（单位是像素）。默认是 *5px*。

[property:function onBeforeRender]

在渲染 LineHTML 之前立即执行的回调。使用以下参数调用此函数：渲染器、场景、相机、几何体、材质、组。

默认情况下，此回调根据线的世界位置和线指向的 HTML元素的位置更新线的几何体。
