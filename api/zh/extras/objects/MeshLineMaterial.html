[name]

绘制线条的特殊材质。旨在与 [page:MeshLine] 和 [page:MeshLineIndexed] 网格一起使用。

Constructor

[name]([page:Material parameters])

[page:Material parameters] - （可选）具有定义材质外观的一个或多个属性的对象。材质的任何属性（包括从 [page:Material] 继承的任何属性）都可以在这里传入。

属性

通用属性查看[page:Material]基类。

[property:Texture alphaMap]

控制沿线不透明度的纹理（需要将 [page:.useAlphaMap useAlphaMap] 设置为 true）。

[property:Number alphaTest]

要执行丢弃的alpha阈值（从0到1）。

[property:Color color]

线条颜色。

[property:Boolean isMeshLineMaterial]

用于检查该类或派生类是否为网格线材质。默认为真。不要更改此设置，因为它在内部用于优化。

[property:Number lineWidth]

线宽（如果 [page:.sizeAttenuation sizeAttenuation] 为真，则为世界单位；否则为屏幕像素）。

[property:Texture map]

控制沿线颜色的纹理（需要将 [page:.useMap useMap] 设置为 true）。

[property:Number opacity]

线条 alpha 值（从 0 到 1）。

[property:Vector2 repeat]

用于缩放[page:.map map]和 [page:.alphaMap alphaMap] UV 坐标的矢量。

[property:Vector2 resolution]

指定画布大小的向量。

[property:Number sizeAttenuation]

使线宽恒定（如果为 0）或不（如果为 1）。

[property:Number useAlphaMap]

使用 [page:.alphaMap alphaMap] 纹理。

[property:Number useMap]

使用 [page:.map map] 纹理。

[property:Number visibility]

通过更改其 alpha 值来控制沿线的部分可见性（从 0 到 1）。
