Морфинг/ 3ds Max

Морфинг - это функция, позволяющая изменять форму объектов в режиме реального времени. Наиболее распространенный вариант использования этой функции - лицевая анимация. Морфинг также хорошо подходит для создания параметрических объектов, которые могут интерактивно изменять свои размеры, форму, толщину, отверстия, профили и т.д.

Настройка

В 3ds Max добавьте модификатор Morpher и настройте каналы для целей морфа. Обратите внимание, что из-за аппаратных ограничений общее количество активных целей морфинга ограничено. Чаще всего это 5 для объектов со скинингом и 6 для объектов без. Дополнительную информацию о расчете ограничения в более сложных случаях см. в этой теме.

Во время рендеринга движок выбирает 6 морф-целей с наибольшим влиянием, игнорируя все избыточные морф-цели.

[anchor:controlling_with_puzzles]

Управление с помощью Puzzles

Для манипулирования объектами морфа в реальном времени есть два пазла: get morph factor и set morph factor. Более подробную информацию смотрите в демонстрации "Параметрические модели".
