Предварительный Просмотр

Пазлы с постобработкой поддерживают функцию live preview. При включенном режиме предварительного просмотра вы можете настроить эффекты постобработки и получить результат в окне просмотра пазлов без необходимости запуска или сохранения/перезагрузки пазлов.

Функция предварительного просмотра в реальном времени включена по умолчанию и поддерживается при отсутствии вводных параментов, подключенных к пазлу:

Перечень Пазлов

[anchor:bloom]

bloom

Включение или изменение параметров эффекта bloom.

Параметры:

  • *threshold* — минимальная интенсивность освещения для рендеринга эффекта bloom.
  • *strength* — интенсивность эффекта bloom.
  • *radius* — радиус размытия эффекта bloom.
[anchor:brightness_contrast]

brightness/contrast

Позволяет настраивать яркость и контрастность рендеринга.

Параметры:

  • *brightness* — коэффициент яркости изображения.
  • *contrast* — коэффициент контрастности изображения.
[anchor:dof]

depth of field

Включение или изменение параметров эффекта глубины резкости (DOF).

Параметры:

  • *focus* — дистанция фокусировки DOF.
  • *aperture* — коэффициент апертуры камеры.
  • *maxblur* — радиус размытия DOF.
  • *depth leak threshold* — порог для устранения артефактов расстояния DOF.
[anchor:grayscale]

grayscale

Включает черно-белый режим рендеринга.

[anchor:ssao]

ambient occlusion

Включение или изменение параметров эффекта окклюзии окружающего пространства экрана (SSAO).

Параметры:

  • *radius* — радиус эффекта SSAO.
  • *min distance* — порог расстояния для уменьшения артефактов, вызванных объектами, расположенными слишком близко.
  • *max distance* — порог расстояния для уменьшения артефактов, вызванных объектами, расположенными слишком далеко.
[anchor:simple_refraction]

simple refraction

Включает эффект легкого отражения (рефракции) для указанного материала.

Параметры:

  • *material* — материал, на который назначается эффект.
  • *distance* — максимальная длина преломленного луча.
  • *render after refraction* — объект(ы), которые будут исключены из очереди обычного рендеринга и будут рендериться после применения эффекта преломления.
[anchor:ssr]

screen space reflection/refraction

Включает эффект отражения/преломления экранного пространства (SSR).

Параметры:

  • *for material* — материал, на который назначается эффект.
  • *steps* — количество шагов поиска глубины вдоль отраженного/преломленного луча (большее значение = лучшее качество, меньшая производительность).
  • *stride* — шаг поиска (в пикселях) (большее значение = более низкое качество, лучшая производительность, 1 пиксель = максимальное качество).
  • *binary search steps* — количество итераций двоичного поиска (большее значение = лучшее качество, меньшая производительность).
  • *thickness* — коэффициент для устранения артефактов SSR (только преломление).
  • *max distance* — максимальная длина отраженного/преломленного луча.
  • *resolution* — множитель разрешения для буфера глубины (от 0 до 1; 1 = наилучшее качество).
  • *jitter* — коэффициент шума для улучшения поиска глубины (от 0 до 1).
  • *render after* — объекты, которые будут рендериться после применения SSR.
