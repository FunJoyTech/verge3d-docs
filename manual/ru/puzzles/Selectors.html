Селекторы

Пазлы из этой категории используются для выбора 3D-объектов, групп, анимационных клипов и материалов для вставки в другие пазлы.

Перечень Пазлов

[anchor:select_object]

select object

Выпадающий список, содержит алфавитный список всех объектов в сцене.

[anchor:all_objects]

all objects

Обозначает все объекты в сцене, чтобы операция могла быть применена к ним в массовом порядке. Не итерируемый.

[anchor:select_group]

select group

Выпадающий список содержит алфавитный список всех групп объектов, представленных в сцене. Операция может быть применена к группе объектов в большом количестве. Не итерируемая.

[anchor:select_animation]

select animation

Выпадающий список содержит алфавитный список всех анимационных клипов, представленных в сцене. Клип анимации называется в соответствии с объектом, к которому применяется анимация в Blender, 3ds Max или Maya.

[anchor:select_material]

select material

Выпадающий список содержит алфавитный список всех материалов, которые присутствуют в сцене.
