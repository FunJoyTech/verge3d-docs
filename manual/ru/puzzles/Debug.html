Перечень Пазлов

[anchor:print_to_console]

print to console

Выводит данные любого типа в консоль браузера. Последнюю обычно можно открыть клавишей F12 (Chrome, Firefox в Windows, Linux). На Mac используйте меню View > Developer > JavaScript Console (Option-Cmd-J) в Chrome или меню Develop > Show Error Console (Option-Cmd-C) в Safari.

[anchor:print_performance_info]

print performance info

Записывает профиль производительности в течение 1 секунды и выводит его в консоль браузера. Последнюю обычно можно открыть клавишей F12 (Chrome, Firefox в Windows, Linux). На Mac используйте меню View > Developer > JavaScript Console (Option-Cmd-J) в Chrome или меню Develop > Show Error Console (Option-Cmd-C) в Safari.

[anchor:get_GPU]

get GPU

Выводит информацию о GPU пользователя - производитель, например: NVIDIA Corporation, Apple Inc., Qualcomm, и модель GPU, например: GeForce GTX 1060 3GB/PCIe/SSE2, Apple A9 GPU, Adreno (TM) 330.
