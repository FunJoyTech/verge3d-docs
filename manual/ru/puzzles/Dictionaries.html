Словари

Пазлы из этой категории позволяют эффективно хранить, передавать и извлекать различные данные. В частности, словарь полезен для хранения множества параметров и их массовой передачи из и в JavaScript и по сети.

Перечень Пазлов

[anchor:create_empty_dict]

create empty dict

Возвращает словарь, не содержащий записей данных.

В JavaScript словарь представлен *Объектом* без свойств, то есть { }.

[anchor:dict_set_key]

dict set key

Устанавливает элемент, который будет связан с указанным ключом в словаре. Ключ должен быть текстом, а присвоенное значение может быть любого типа (текст, число, список, другой словарь и т.д.).

[anchor:dict_get_key]

dict get key

Возвращает элемент, связанный с указанным ключом в словаре.

[anchor:get_keys]

get keys

Возвращает список всех ключей, присутствующих в словаре.

[anchor:dict_check_key]

dict check key

Проверяет, присутствует ли указанный ключ в словаре, и возвращает результат с булевым типом *true* или *false*.

[anchor:dict_remove_key]

dict remove key

Удаляет элемент, связанный с указанным ключом в словаре, и удаляет сам ключ.

[anchor:is_empty]

is empty

Проверяет, не содержит ли указанный словарь ключей, и возвращает результат с булевым типом *true* или *false*.
