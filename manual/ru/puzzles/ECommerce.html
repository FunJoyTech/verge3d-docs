[anchor:woo_connect]

connect to WooCommerce

Подключитесь к системе WooCommerce, чтобы начать получать информацию о продукте.

Посмотрите этот урок для получения подробной информации.

[anchor:woo_product_info]

product info

Возвращает словарь с информацией о продукте, полученной с помощью команды *connect to WooCommerce*. Этот словарь (объект JavaScript) содержит следующие поля:

  • *price* - цена для выбранного клиентом варианта продукта, например, 12.
  • *quantity* - количество выбранных клиентом товарных позиций, например, 2.
  • *sku* - складская единица для выбранного клиентом варианта продукта.
  • *name* - название товара, которое будет отображаться в корзине, например, "Спиннер - желтый, большой".
  • *attributes* - словарь с парами имен/значений атрибутов для выбранного клиентом варианта продукта, например {цвет: "Желтый", размер: "Большой"}.
  • *weight* - физический вес продукта для выбранного клиентом варианта продукта.
  • *length*, *width*, *height* - физические размеры продукта для выбранного клиентом варианта продукта.
  • *type* - тип товара, указанный в панели администратора WooCommerce, один из следующих: "variable", "simple", "group" или "external".
  • *children* - доступна только для "групповых" продуктов. Включает информацию по каждому субпродукту.
  • *status* - ответ сервера: "ok" или "error".

[anchor:place_order]

place order

Составляет скрытую форму заказа с полями "title", "contents" и"total price", необязательным скриншотом и отправляет эту форму на указанный URL в соответствии со спецификацией, описанной в разделе Wordpress Plugin. По умолчанию форма заказа отправляется на страницу заказа демо-версии.

Verge3D поставляется с бесплатным плагином Wordpress, который способен обрабатывать запросы, отправленные этим пазлом. При получении такого запроса этот плагин Wordpress отображает страницу с расширенной формой, дополненной полями контактов и комментариев, встроенным скриншотом и капчей. В итоге заполненная форма отправляется клиентом, и в админ-интерфейсе Wordpress создается новый заказ. И клиент, и менеджер по продажам получают уведомление о заказе по электронной почте.

Информацию о настройке смотрите в разделе Плагин Wordpress данного руководства.

Обязательно включите скриншоты в пазле configure application, иначе скриншот может отображаться черным цветом.
