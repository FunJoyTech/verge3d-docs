3D-посты Facebook

Функция 3D-постов в Facebook теперь устарела. Для встраивания приложений Verge3D используйте Facebook Apps.

Создать 3D-поста на Facebook очень просто: сначала вы выполняете экспорт в формате .glb (.glb - это двоичная версия glTF), а затем перетаскиваете этот файл в композитор поста на Facebook. Ниже приведена подробная схема.

  • Экспорт GLB из Blender

  • Начальные файлы

#

Экспорт GLB из 3ds Max

Чтобы выполнить экспорт в 3ds Max, перейдите в меню Verge3D и выберите "Export Facebook GLB...".

В появившемся диалоговом окне файла выберите путь экспорта и нажмите *Сохранить*.

#

Экспорт GLB из Blender

Чтобы выполнить экспорт в Blender, перейдите в "File > Export" и выберите опцию "Facebook GLB (.glb)". Общий вариант GLB-экспорта "Verge3D glTF Binary (.glb)" также подойдет, но с его помощью на выходе мы получим немного менее оптимизированные файлы.

#

Составить 3D Post в Facebook

Теперь вы можете просто перетащить файл GLB в свой пост на Facebook.

В композиторе поста Facebook вы также можете выбрать фон по своему вкусу.

Посмотрите, как это выглядит на Facebook: ссылка. Исходный файл для этой модели можно найти в Verge3D в папке "applications" (см. ниже в разделе "Starter files").

#

Начальные Файлы

Мы предоставляем образец проекта Verge3D, в котором настроены необходимые параметры экспорта как для 3ds Max, так и для Blender. Стартовый проект содержит точно такую же карту окружения, которая используется в Facebook для максимальной согласованности.

Его можно найти внутри дистрибутива Verge3D, расположенного в "applications/fb_max" (3ds Max) или "applications/fb" (Blender).

#

Технические Требования

  • Facebook требует, чтобы ваш GLB файл не превышал 3 Мб в целом, чем меньше, тем лучше.
  • Поддерживаются только glTF-совместимые PBR-материалы (3ds Max, Blender) - подумайте о конвертации ваших материалов.
  • Для текстур должны использоваться файлы PNG или JPEG, которые должны иметь размерность, кратную 2 (например 256x512).
  • Текстуры могут достигать 4k, хотя разработчики Facebook рекомендуют не превышать 2k.

#

Подсказки касательно Оптимизации

Вот несколько рекомендаций о том, как сделать так, чтобы ваш файл GLB не превышал 3 Мб и при этом имел достойную графику.

  • Предпочитайте текстуры JPEG, а не PNG, если вам не нужна прозрачность.
  • Рассмотрим поиск оптимальной степени сжатия для файлов JPEG.
  • Очистите файлы ассетов от всех неиспользуемых данных. Сюда входят вспомогательные объекты, ключи анимации и ключи формы (которые в настоящее время не поддерживаются Facebook), неиспользуемые цвета вершин, UV и т.д.
