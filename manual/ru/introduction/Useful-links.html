Помощь с API

Просто погуглите «Как сделать ... в three.js», и, с вероятностью 99%, вы сможете сделать то же самое с Verge3D, поскольку мы пытаемся поддерживать совместимость API между двумя фреймворками.

[anchor:browser_technologies]

Технологии браузера — WebGL, WebXR и т.д.

  • [link:https://www.soft8soft.com/webgl-supported-browsers-and-troubleshooting/ WebGL supported browsers and troubleshooting] - это ваша аптечка на случай возникновения проблем с поддержкой WebGL.
  • [link:https://www.soft8soft.com/webglreport/ WebGL System Report] — проверьте, на что способна ваша система.
  • [link:https://developer.mozilla.org/en-US/docs/Web/API/WebGL_API/WebGL_best_practices WebGL Best Practices] — рекомендации по работе с WebGL
  • [link:https://www.soft8soft.com/gamepad_diagnostics/gamepad_diagnostics.html Gamepad Diagnostics Tool] — инструмент интерактивной диагностики Gamepad API. Проверьте, совместимо ли ваше устройство. Узнайте о модели, осях, кнопках и получите иную информацию для использования с Verge3D.
  • [link:https://immersive-web.github.io/webxr/ WebXR Device API] — официальная спецификация WebXR.
  • [link:https://immersive-web.github.io/webxr-samples/ WebXR Samples] — официальные примеры WebXR включают демонстрации как AR, так и VR.
