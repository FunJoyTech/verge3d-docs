App Manager (менеджер проектов)

App Manager предназначен для создания и выполнения различных операций с приложениями Verge3D.

Содержание

[contents] [anchor:Running_App_Manager]

Запуск App Manager

App Manager — это классический сайт, отображаемый локальным веб-сервером, наподобие панели настроек WiFi роутера.

Доступ осуществляется по адресу localhost:8668. Вы можете переходить со страницу на страницу, обновлять их или добавлять в закладки. Сам сервер запускается несколькими способами (см. ниже).

[anchor:From_Blender_Max_Maya]

Вариант 1. Прямой доступ из Blender, 3ds Max или Maya

При старте вашего 3D редактора происходит автоматический запуск локального веб-сервера, если, конечно, установлен плагин Verge3D. Щелкните по кнопке App Manager для Blender или выберите соответствующий пункт меню для других редакторов. Это действие откроет App Manager в веб-браузере по умолчанию.

В случае неработоспособности веб-сервера (такое может случится, например, если вы закроете первый запущенный экземпляр 3D-редактора) данная функция его перезагрузит.

[anchor:Using_Desktop_Application]

Вариант 2. Использование классического приложения

Локальный веб-сервер можно запустить без Blender, 3ds Max или Maya. Просто щелкните значок на рабочем столе, созданный установщиком Verge3D (только для Windows). Страница App Manager откроется в веб-браузере.

Если веб-сервер уже запущен (например из 3D-редактора), данное приложение просто откроет App Manager в браузере.

[anchor:From_Command_Line]

Вариант 3. Работа с командной строкой

Установите Python 3, например из Microsoft Store. Затем запустите * Command Prompt * и введите:

cd verge3d_blender

(укажите корректный путь к директории, где установлен Verge3D). Для версии Blender введите:

python manager\server.py BLENDER

для Max:

python manager\server.py MAX

для Maya:

python manager\server.py MAYA

Откройте веб-браузер и перейдите по адресу localhost:8668. Пользователи macOS или Linux могут использовать одни и те же команды, но только с обратным слэшем (/).

Страница приложений

[anchor:List_Main_Page]

Список на главной странице

После запуска или обновления главной страницы, App Manager сканирует директорию *applications* и отображает все найденные проекты в виде списка. Если в настройках программы указан *External applications directory* (внешний каталог приложений), то папки из него добавляются в конец списка.

Для каждого приложения из списка App Manager демонстрирует все html, gltf/glb файлы, а также проекты 3D редакторов. Эти файлы можно просмотреть, щелкнув по нужной иконке.

Если приложение содержит большое количество файлов, то они скрываются для удобства просмотра. Щелкните по иконке треугольника, рядом с названием приложения, и вы получите полный доступ.

Используйте строку поиска для фильтрации приложений.

Страница приложения

Кликните по названию приложения, чтобы перейти на его страницу.

Здесь перечислены все файлы, которые доступны для запуска. Вы также можете фильтровать приложения, используя строку поиска.

[anchor:Creating_Applications]

Создание приложения

Новое приложение создается с помощью кнопки на левой панели.

Введите имя для нового приложения и выберите шаблон. По умолчанию предлагается 3 варианта: стандартный светлый или темный, а также на основе кода. Вы можете создавать и добавлять дополнительные шаблоны .

Вы также можете добавить дополнительные модули в приложение, чтобы активировать поддержку Internet Explorer 11, обеспечить совместимость с API WebVR или включить физический движок.

После завершения генерации, будет показана ссылка на новое приложение.

Созданное приложение появится в общем списке.

Публикация

Вы можете опубликовать приложение, щелкнув по нужной кнопке в главном окне:

...или по ее аналогу на правой панели в окне приложения.

Начнется загрузка приложения в Verge3D Network. Это позволит получить к нему доступ онлайн (действие может быть отменено).

Если загрузка файлов прекратилась с выводом сообщения «The difference between your local system time and the server time is too large», то установите корректное время на своем компьютере и попробуйте снова.

После завершения загрузки будет показано окно с прямой ссылкой на приложение, а также код для встраивания в уже имеющиеся страницы.

Вы также можете использовать кнопки социальных сетей и генератор QR-кода.

Загруженные файлы можно просмотреть или удалить на странице Network Directory.

[anchor:Upload_Archive]

Загрузка архива

Вы можете загрузить приложение в виде архива zip, используя кнопку на правой панели на странице приложения.

Программа сожмет ваши файлы и загрузит архив в Verge3D Network (операция может быть отменена). Используйте эту возможность для передачи контента вашим коллегам, клиентам или другим разработчикам Verge3D.

Если загрузка файлов прекратилась с выводом сообщения «The difference between your local system time and the server time is too large», то установите корректное время на своем компьютере и попробуйте снова.

После завершения загрузки откроется окно со ссылкой на zip-архив.

Загруженные файлы также могут быть просмотрены и удалены в окне Verge3D Network.

Шаблоны

[anchor:Whats_a_Template]

Что такое шаблоны

При создании нового приложения вы можете выбрать шаблон, на основе которого будет создано приложение. По умолчанию предлагается 3 шаблона: "Standard Light", "Standard Dark", and "Code-Based".

Исходные файлы для этих шаблонов можно найти в папке: "verge3d_blender\manager\templates". Вы можете изменить эти файлы или создать совершенно новый шаблон для использования в своем проекте.

В папке шаблонов также есть скрытый шаблон под названием «Embeddable», который вы можете использовать для создания приложений React.js или Vue.js.

Есть много способов изменить или улучшить ваш шаблон. Вы можете поменять макет, отредактировав файлы HTML/CSS, используя некоторый код или библиотеку JavaScript. Заменить элементы пользовательского интерфейса или прелоадер, добавив несколько пазлов, которые будут включены в каждое новое приложение по умолчанию.

[anchor:Creating_Templates]

Создание шаблонов

Чтобы создать свой шаблон, нажмите кнопку *Manage App Templates* на левой панели:

На открывшейся панели кликните +Template и укажите имя для нового шаблона (например, "My Template"), затем нажмите *Apply*:

В дальнейшем, при создании нового приложения, будет появляться возможность выбрать свой шаблон:

По умолчанию для вашего приложения будет создана пустая папка. Самый простой способ продолжить - это скопировать один из существующих шаблонов (например, "Standard Dark") и переименовать дубликат в соответствии с именем вашего шаблона (т.е. "My Template").

[anchor:App_Manager_Settings]

Настройки менеджера приложений

Нажав кнопку шестеренки на левой панели, вы можете открыть панель с различными настройками:

По умолчанию это выглядит так:

Notify about Verge3D updates
Диспетчер приложений отобразит уведомление, если доступна новая стабильная версия Verge3D (опция по умолчанию активна).
Upload app sources (models, puzzles, backups)
Функция Publish загрузит файлы, которые обычно не нужны для запуска приложения в сети Интернет и поэтому пропускаются по умолчанию. Сюда входят .blend, .max, .ma/.mb, visual_logic.xml и содержимое папки v3d_app_data. Включите опцию, если хотите поделиться исходными файлами со своими коллегами.
Enable external server interface
Используйте IP-адрес в Wi-Fi или локальной сети вместо localhost: 8668 . Вы можете найти IP-адрес своего компьютера в настройках операционной системы: Win 10 , Win 7 , Mac , Linux . Запустите менеджер приложений, перейдя к сетевому адресу, например: 192.168.1.15:8668
Verge3D Network cache age (minutes)
Устанавливает период времени, по истечении которого Verge3D Network будет обновлять повторно загруженное приложение. Установите значение 0 (мгновенное обновление) для более быстрых итераций в разработке.
External applications directory
Использовать указанный абсолютный путь (например, C:\Users\Alex\ my_verge) для папки приложений вместе со стандартной (например, verge3d_blender\applications). Папки из указанного каталога будут отображаться в списке приложений менеджера приложений.
Current theme
Выбор темы (светлый и темный режимы) для менеджера приложений и редактора пазлов.
