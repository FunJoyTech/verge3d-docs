Обзор системы материалов / Blender

Verge3D поддерживает следующие типы материалов:

[contents] [anchor:Eevee]

Eevee Материалы

Eevee это движок для рендеринга в реальном времени, который мы рекомендуем использовать с Verge 3D.

Вы можете ознакомиться более подробно с нодами, которые поддерживает Eevee, пройдя по следующей ссылке.

[anchor:Cycles]

Cycles Материалы

Cycles это движок рендеринга с поддержкой трассировки лучей в Blender, он может быть воссоздан на движке рендеринга в реальном времени Verge3D только приблизительно. Большинство нод Cycles взаимозаменяемы нодами Eevee, поэтому вы можете трансформировать существующие ассеты, основанные на Cycles, без особых препятствий.

[anchor:glTF_compliant_PBR]

PBR Материалы совместимые с форматом glTF

Данные материалы соответствуют glTF 2.0 стандарту для физически корректных материалов (PBR). Согласно этому стандарту, информация о материале закодирована в наборе текстур: base color, occlusion-roughness-metallic упакованные в R, G и B каналы, соответственно.

В Verge3D, PBR материалы совместимые с форматом glTF отображаются с помощью шейдерной ноды Principled BSDF, ко входам которой, подключаются PBR текстуры.

Кроме того, вы можете настроить ноды world в качестве окружающей среды, хотя это не является glTF 2.0 стандартом.

Ознакомьтесь с главой glTF Materials для большей информации как настроить PBR материалы совместимые с форматом glTF.
