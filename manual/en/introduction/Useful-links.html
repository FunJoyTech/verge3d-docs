Browser Technologies — WebGL, WebXR etc

  • [link:https://www.soft8soft.com/webgl-supported-browsers-and-troubleshooting/ WebGL supported browsers and troubleshooting] — this is your first aid kit in case of any issues with WebGL support.
  • [link:https://www.soft8soft.com/webglreport/ WebGL System Report] — check what your system is capable of.
  • [link:https://developer.mozilla.org/en-US/docs/Web/API/WebGL_API/WebGL_best_practices WebGL Best Practices] — recommended ways to use WebGL.
  • [link:https://www.soft8soft.com/gamepad_diagnostics/gamepad_diagnostics.html Gamepad Diagnostics Tool] — Interactive Gamepad API diagnostics tool. Check if your device is compatible and retrieve the model, axes, buttons and other information for use with Verge3D.
  • [link:https://immersive-web.github.io/webxr/ WebXR Device API] — official WebXR specification.
  • [link:https://immersive-web.github.io/webxr-samples/ WebXR Samples] — official WebXR examples, include both AR and VR demos.
